# learn_deep

Learn deep 这个阶段的目标是在移动设备上运行`tensorflow lite`。ld项目更加关注工程实践，算法方面只会有一些必要的内容。

- tensorflow lite c++ (a or lib)
- object detection
- android nativelibrary example

# 过程计划

- build the lite
- docker化lite的编译
- make a library
- link lite to library
- build the library c++
- 向量加法的模型
- 冻结模型
- library 加载模型
- 创建安卓App项目
- 创建安卓native library项目
- 为native library引入library
- 测试安卓App运行向量加法
